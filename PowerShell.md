## Basic PowerShell Commands:

1. **cd ML**
   -> Go to the "ML" folder.

2. **cd ..**
   -> Move up one directory level.

3. **ls**
   -> List files and folders.

4. **ls -Recurse**
   -> List files and folders, including all subdirectories.

5. **ls -Hidden**
   -> Show hidden files and folders.

6. **pwd**
   -> Display the current working directory.

7. **md NewFolder**
   -> Create a new folder.

8. **ni NewFile.txt**
   -> Create a new file.

9. **mv New.txt C:\...**
   -> Move a file to another location.

10. cp file_to_copy C:\new_location
    -> Copy a file to another location.

11. **del New.txt**
    -> Delete a file.
12. **Clear-Host**
    -> Clear the console. (Shortcut: Ctrl + L)
